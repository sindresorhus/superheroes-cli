# superheroes-cli

> Get superhero names

## Install

```sh
npm install --global superheroes-cli
```

## Usage

```
$ superheroes --help

  Examples
    $ superheroes
    Spider-Ham

    $ superheroes --all
    3-D Man
    A-Bomb
    ...

  Options
    --all  Get all names instead of a random name
```
